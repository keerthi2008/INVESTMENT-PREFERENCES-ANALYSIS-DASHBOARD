# INVESTMENT-PREFERENCES-ANALYSIS-DASHBOARD

investment preferences analysis dashboard using excel visulation.

This project analyzes participants' investment preferences using Excel.

The dashboard visualizes gender distribution, savings objectives, investment sources, and duration patterns using charts and pivot tables.

## Tools Used:
- Microsoft Excel.
  
- Pivot Tables'
  
- Data Visualization (Pie, Bar, Line, Bubble charts).

## Insights:
- Majority of participants prefer equity investments.
  
- Savings objectives vary across participants.
  
- Different investment durations are observed among age groups.
